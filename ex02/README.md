# 第２回
##　電卓(ex02/calc.py)
### 追加機能
- 四則演算：+以外の四則演算をする
- クリア：entryに入力されている数式をdeleteする。
- 2進数ボタン：入力されている数値を10進数から2進数にする。
- 8進数ボタン：入力されている数値を10進数から8進数にする。
- 16進数ボタン：入力されている数値を10進数から16進数にする。
